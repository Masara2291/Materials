使用するのは、キーボード(テンキー含む)・マウスです

WASD : 移動
QE : 90度毎に向き変更
UJ : オブジェクトの高さ変更
OL : オブジェクトの座標を変更
G : グリッドの使用モード切替
1～0 : オブジェクト選択(1～10種類分)
テンキー1～0 : オブジェクト選択(11～20種類分)
Lcontrol + 1～0 : オブジェクト選択(21～30種類分)
Lcontrol + テンキー1～0 : オブジェクト選択(31～40種類分)
左クリック : オブジェクト消去
右クリック : オブジェクト設置
マウス : 視点操作
ホイール : 高さ変更(有線以外だと動作不良有り)
space : カメラの高さを上昇
Lshift : カメラの高さを下降
control + B : 変更内容をテキストファイルに書き出す
control + P : オブジェクトを全て消去
control + S : オブジェクト座標保存
control + C : オブジェクトの種類・向きを取得
control + X : 選択中オブジェクトを切り取り
control + V : 取得した情報で設置
control + Z : 操作を戻す
control + Y : 戻した操作を戻す
F1 : 選択中オブジェクトの座標を設置時の座標に戻す
F2 : 選択中オブジェクトの向きを設置時の方向に戻す
F3 : オブジェクトを設置するモードに変更
F4 : 敵を設置するモードに変更
X / Z : 軸固定
P : ポーズ切り替え

読み込むテキストファイルは「data/TXT/LoadTextName.txt」の中身を変更してください
「.txt」ファイルは作っていなくても動作しますが、
「.csv」ファイルは作っているファイルを指定してください

ブロック・敵のオブジェクト情報を変更する場合、
ブロックの場合は「data/TXT/blockmodel.txt」、敵の場合は「data/TXT/enemymodel.txt」
の中身を変更してください
xファイルはブロック・敵各40種類まで読み込ませる事が出来ます

注意:アプリケーション実行からオブジェクトが表示されるまでの間にマウスを動かした場合、
カメラが止まらなくなることがあるので、その時はエンターキーで初期化してください
